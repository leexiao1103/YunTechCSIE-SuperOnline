# YunTechCSIE-SuperOnline
這個APP是我在系學會舉辦的全校真人實境RPG寫的，可以讓參加者登入帳號，登入會會有每組的隊伍資訊，並能在遊戲中慢慢增加道具和技能，還有建立聊天室可以和其他隊伍聊天。
